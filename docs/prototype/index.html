设计系统 总览

所有页面共享同一组设计 token：深墨底 + 铜金渐变 + Serif 标题 + Mono 数据字体 + 状态点 + 场记板等电影符号。

布局原则

  • · 240px 左侧栏承载项目/导航/设置入口
  • · 主区 56px 顶栏承载面包屑 + 状态 + Tab 切换
  • · 主区滚动，侧栏固定（永不消失的上下文）
  • · 右侧 Agent 栏仅工作台出现（创作时永远在线）
  • · 详情用抽屉而非新页面（保留上下文）
  • · 宽 1440 / 100%响应式，主断点 1080 / 860

关键组件

  • · Project Card · 16:9 海报 + 状态徽章 + 进度
  • · Episode Row · 集号 + 标题 + 进度条 + 状态
  • · Pipeline Step · 垂直时间线 + 三态视觉
  • · Take Timeline · 水平滚动镜头带（Premiere 风格）
  • · Storyboard Card · 首/末帧并排 + 进度 + 快捷
  • · Agent Message · 用户/AI 对话 + 工具调用轨迹
  • · Provider Card · Logo + 优先级 + 一键测
  • · Asset Drawer · 右滑详情 + 内嵌播放

交互模式

  • · ⌘K 全局搜索 · 项目 / 角色 / 场景统一入口
  • · Enter / Space · 卡片键盘可达性
  • · / 唤起指令 · Agent 聊天快捷操作
  • · 悬停预览 · 视频 hover 静音播放 2s
  • · 拖拽排序 · 镜头重排 / 角色重排
  • · 批量操作 · 多选资产 → 批量导出 / 删除
  • · 离线感知 · 网络断开时显示 banner + 本地草稿
  • · 实时协作 · 同时显示其他协作者头像
